    Does this connect to Gemini today?

    This MVP includes the frontend workflow and API-ready data structures. The next step is adding Next.js API routes and Gemini credentials.

    Can users save recipes?

    Yes. Saved recipes are stored locally in the browser in this first version.

    What is the north star metric?
